# Code of Conduct

IONe adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

## Scope

Applies to all project spaces: this repository, issue tracker, pull requests, discussions, and any official project channels.

## Enforcement

Report issues to [morton@myma.us](mailto:morton@myma.us). Maintainers will respond within 7 days and treat reports confidentially.

## Summary

Be respectful. Focus on what is best for the project and the community. Unacceptable behavior includes harassment, trolling, derogatory comments, personal attacks, and sustained disruption of discussion. Maintainers may remove, edit, or reject contributions that don't align with this standard.
